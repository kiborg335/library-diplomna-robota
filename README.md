# Бібліотека книг

Десктопний застосунок для ведення особистої бібліотеки з доступом до каталогу Project Gutenberg.

**Кваліфікаційна робота** на підтвердження ступеня фахового молодшого бакалавра

**Навчальний заклад:** [ВСП «ППФК НТУ «ХПІ»](https://www.polytechnic.poltava.ua/)

**Спеціальність:** 121 Інженерія програмного забезпечення

**Керівник роботи** — Олійник Владислав Васильович

**Автор** — Базарбаєв Артем Шухратович

**Рік виконання:** 2026

---

## Завдання до дипломної роботи

Розробити десктопний застосунок бібліотеки книг, який забезпечує:

- імпорт каталогу книг із Project Gutenberg (понад 65 000 книг)
- пошук книг за назвою та автором
- фільтрацію за мовою та статусом завантаження
- завантаження тексту книг із серверів Gutenberg
- читання книг у вбудованій читалці
- додавання власних книг у форматах PDF, Word, TXT
- локальне зберігання даних у базі SQLite
- роботу без постійного підключення до інтернету
- простий україномовний інтерфейс

---

## Використані технології

- **C#** — мова програмування
- **Windows Forms** — графічний інтерфейс
- **SQLite** — локальна база даних
- **CsvHelper** — парсинг CSV-файлу
- **HttpClient** — завантаження даних
- **.NET Framework 4.8** — платформа
- **Visual Studio** — середовище розробки
- **GitHub** — контроль версій
